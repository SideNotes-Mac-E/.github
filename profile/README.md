## Overview

SideNotes Advanced is a quick-access note application from Apptorium that presents a note panel sliding from the edge of your screen, providing instant access to notes, code snippets, checklists, and reference text without disrupting your current workflow or switching to a separate application window. It occupies the unique space between heavyweight note applications that require context switching and clipboard managers that lack organization, providing a persistent, organized scratchpad that is always one gesture away. The application is designed for the kinds of quick notes and reference snippets that do not warrant opening a full notes application.

The slide-out panel appears and disappears with a customizable hotkey or screen edge gesture, keeping notes accessible without permanently consuming screen space or requiring window management. The panel width is adjustable, and the slide direction can be configured for left or right screen edges depending on preference and monitor arrangement. Auto-hide behavior closes the panel when focus moves to another application, keeping the workspace clean.

Note organization with folders, colors, and pinning keeps the panel organized without imposing the heavyweight structure that full notes applications require. Color-coded notes provide visual categorization that makes different note types immediately distinguishable. Pinned notes remain at the top of the panel regardless of creation order, keeping the most important references always visible.

Code snippet support with syntax highlighting for common programming languages makes SideNotes practical for developers who need to keep code fragments, terminal commands, configuration snippets, and API references immediately accessible while working. Markdown support renders formatted text for notes that benefit from structure beyond plain text. Images and file attachments can be embedded in notes for visual reference material.

For anyone who frequently writes quick notes, saves code snippets, or keeps reference text that needs to be accessible without opening a separate application, SideNotes Advanced is the note tool that lives at the edge of your screen and appears only when you need it. Its minimal footprint and instant accessibility make it the fastest path from thought to written note.

## Performance

SideNotes balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
